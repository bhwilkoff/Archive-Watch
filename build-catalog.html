Build the tvOS app's seed catalog.

This page reads featured.json, resolves every dynamic shelf against Archive.org, fetches per-item metadata, and (if you provide a TMDb bearer token) enriches each result with poster art and credits. The output is catalog.json, which ships inside the tvOS app bundle so the app has polished content on first launch without any network calls.

Runs entirely in your browser. Nothing leaves this page except the direct API calls to archive.org and (optionally) themoviedb.org.

Configuration

Without a token, items get Archive-only metadata (titles + durations + tags) and the app will enrich them from TMDb at runtime. With a token, the catalog ships fully enriched and first-launch needs zero network. The token is stored in this browser's localStorage, never transmitted anywhere except directly to TMDb.
